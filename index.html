Habilidades

Portafolio

Sistema Contable.

Sistema contable desarrollado para la gestión integral de estados financieros, que abarca el libro diario, el libro mayor, el balance general, la balanza de comprobación y el estado de resultados. Este sistema está implementado en .NET Core con C# y utiliza MySQL como base de datos, ofreciendo una solución robusta y escalable para el manejo contable.

Sistema Facturación

Sistema de facturación electrónica y gestión de notas de crédito, con control integral de clientes, desarrollado en .NET Core con C# y MySQL. Esta solución ofrece una administración eficiente y segura de facturas electrónicas y notas de crédito, optimizando el manejo de las relaciones con los clientes y garantizando una integración robusta y escalable.

Sistema de Emprendimiento

Sistema para emprendimiento que permite el control de servicios para diversos usuarios y la gestión de publicidad con controladores de pago. Este sistema está desarrollado con Angular, Node.js y utiliza APIs REST, proporcionando una solución completa y escalable para la administración de servicios y publicidad.

Odontograma

Participé en el desarrollo de un sistema que incluye un odontograma con control completo de clientes, abarcando todas las operaciones posibles relacionadas con su gestión. Este sistema permite un seguimiento detallado y eficiente de los tratamientos y procedimientos odontológicos, facilitando una administración integral y precisa de la información del paciente.

Sistema web de control médico de pacientes

Colaborador en el desarrollo de una plataforma web privada para el seguimiento de consultas por paciente. En este proyecto, trabajé en el desarrollo del aplicativo utilizando Angular para el frontend y Node.js para el backend

ERP

Desarrollador de un sistema ERP personalizado, que abarca operaciones clave en diversas áreas de la organización e incluye funcionalidad para facturación electrónica. Este proyecto fue desarrollado utilizando Angular para el frontend, Node.js para el backend y SQL Server como gestor de bases de datos, ofreciendo una solución completa y adaptada a las necesidades específicas de la empresa.

Sitio Fundación

Desarrollador del sitio oficial de fundación, incluyendo también el desarrollo de la sección administrativa para gestionar y actualizar la información del sitio. El proyecto fue implementado utilizando PHP para el sitio web público y Angular para la interfaz administrativa, proporcionando una solución completa y eficiente para la visualización de contenido.
